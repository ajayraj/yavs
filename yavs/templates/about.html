Welcome to Yet Another Video Site!



About this site:


YAVS is built with Python using Django for the backend

Text transcription is done with SpeechRecognition using Google's engine. Feature currently only works for English content.

If unable to transcribe, audio is assumed to be music/too noisy to automatically categorize.

Processing, training, and classification were done using Spacy and sklearn's SVM's.

Classification works across the following 11 categories:

Comedy, Fashion, Food, Gaming, Kid's Entertainment, Lecture/Learning, News, Reviews, Science, Sports, Tech

In progress: an API to send transcribed text queries to in order to classify using my trained model.
